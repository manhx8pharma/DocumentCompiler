🔄 Luồng dữ liệu chính
- UI: người dùng thao tác (CRUD workflows, mappings, documents, fields).
- API Gateway: xác thực, phân quyền, điều phối request.
- Workflow Manager: quản lý quy trình & steps.
- Template Repository: cung cấp metadata fields.
- Mapping Engine: xử lý ánh xạ auto/manual, transform, conflict.
- Prefill Engine: điền dữ liệu từ bước trước sang bước sau.
- Document Service: lưu DocumentInstance & FieldValues.
- Provenance Service: ghi nguồn gốc dữ liệu.
- Audit Log Service: ghi lại mọi hành động CRUD.
- Database Layer: lưu trữ toàn bộ dữ liệu & quan hệ.

🎯 Ý nghĩa
- Frontend: tập trung vào trải nghiệm người dùng.
- Backend Services: phân tách rõ chức năng, dễ mở rộng.
- Database: nền tảng lưu trữ, đảm bảo toàn vẹn dữ liệu.
- Audit & Provenance: minh bạch, kiểm soát, tuân thủ.
- Luồng dữ liệu: rõ ràng từ UI → API → Services → DB.
